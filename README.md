# Trabajo-Fin-de-Grado
Este repositorio contiene el código, figuras y documentación generados como parte del Trabajo Fin de Grado, cuyo objetivo ha sido estudiar la inferencia y comparación de redes de regulación génica dinámicas en leucemia linfocítica crónica. Se han utilizando datos transcriptómicos longitudinales del estudio de referencia de Marku et al.
